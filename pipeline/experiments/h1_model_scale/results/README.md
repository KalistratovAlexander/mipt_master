# H1 — результаты оценки

Числа, приведённые в таблицах H1 диссертации, получены из этих артефактов.

- **`h1_summary.json`** — компактная сводка по каждому размеру (0.6B/1.7B/4B/8B): Recall@10 по уровням иерархии (A/AB/ABC/ABCD) для всех задач предсказания идентификатора, `cosine_sim` для задач генерации текста, перплексия WikiText-2 (база и после дообучения).
- **`ppl_results.json`** — базовая и дообученная перплексия WikiText-2 по всем четырём размерам (подтверждает катастрофическое забывание, в т.ч. «1.7B ×158»).

Полные per-sample артефакты оценки (лучевые предсказания, поэлементный hit@10, текстовые метрики) по всем четырём размерам опубликованы как датасет на HuggingFace:

https://huggingface.co/datasets/kalistratov/h1-eval-results-4size

Воспроизведение оценки и статистики: `bash pipeline/experiments/h1_model_scale/run_h1.sh` (единый оценщик + `stat_tests.py`, парный бутстреп с поправкой Бонферрони m=3).
