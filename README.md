## Аннотация

В данной работе исследуется применение иерархических семантических идентификаторов (Semantic IDs) для генеративных товарных рекомендаций. Каждому товару каталога присваивается уникальный многоуровневый код, полученный с помощью Residual Quantized VAE (RQ-VAE), что позволяет большой языковой модели генерировать идентификаторы товаров непосредственно как последовательности токенов. Подход оценивается на датасете Amazon Pet Supplies (63 319 товаров, 101 926 пользовательских последовательностей) путём сравнения двух масштабов модели — Qwen3-1.8B и Qwen3-8B — при идентичных условиях обучения.

## Метод

Система состоит из трёх этапов:

1. **Назначение Semantic ID.** Эмбеддинги товаров (Qwen3-Embedding-0.6B, 1024d) квантуются трёхуровневым RQ-VAE с кодовыми книгами размером 256 и rotation trick для оценки градиентов. Детерминированный четвёртый уровень (D) устраняет коллизии внутри групп (A,B,C), обеспечивая уникальные 4-уровневые идентификаторы для всех 63 319 товаров.

2. **Расширение словаря.** В токенизатор LLM добавляются 1 027 специальных токенов (`<|sid_start|>`, `<|sid_end|>`, `<|A0|>`–`<|D255|>`). На данном этапе обучаются только матрицы эмбеддингов (Stage 1).

3. **Полное дообучение.** LLM обучается на 4,7 млн диалогов в формате инструкций, охватывающих задачи сопоставления текста и SID, последовательных рекомендаций и предсказания совместных покупок (Stage 2). Обучение использует Unsloth для загрузки модели, маскирование инструкций, упаковку последовательностей (~3x ускорение) и cosine-with-minimum-LR планировщик.

Обе модели обучаются с идентичными гиперпараметрами (LR=2×10⁻⁵, warmup 3%, weight decay 0.01, эффективный батч 128, 3 эпохи) для контролируемого сравнения.

## Результаты

Точность предсказания Semantic ID (совпадение на уровне A, жадное декодирование):

| Задача | Qwen3-1.8B | Qwen3-8B | Δ |
|--------|-----------|----------|---|
| Текст → SID | 59.9% | 64.6% | +4.7 |
| Последовательная рекомендация | 7.0% | 9.7% | +2.7 |
| Предсказание совместных покупок | 5.5% | 5.8% | +0.3 |

Оценка: 3 000 примеров на задачу, 11 типов задач, NVIDIA H100 80GB.

## Воспроизведение

### Окружение

```bash
pip install -r requirements.txt
```

Для обучения на GPU необходимо установить PyTorch с поддержкой CUDA:
```bash
pip install torch --index-url https://download.pytorch.org/whl/cu121
```

### Подготовка данных

Ноутбуки выполняются последовательно от `data/download_data.ipynb` до `pipeline/prepare_data_for_llm.ipynb`. Шаги 2–3 требуют ключ Gemini API.

### Обучение

Скрипты обучения предназначены для развёртывания на [vast.ai](https://vast.ai) с GPU NVIDIA H100 80GB.

```bash
# Сборка пакета для развёртывания (масштаб: 0.6b | 1.8b | 4b | 8b)
bash pipeline/fine_tune_h1/pack.sh 8b

# Загрузка на удалённый GPU-инстанс
scp vast_8b_package.tar.gz root@<HOST>:/workspace/

# На удалённом инстансе
cd /workspace && tar xf vast_8b_package.tar.gz
export HF_TOKEN=hf_...
bash run_smoke.sh     # smoke-тест (~10 мин)
bash stage1/run.sh    # Stage 1: ~1 час
bash stage2/run.sh    # Stage 2: ~10 часов
```

Остальные масштабы собираются той же командой с аргументом `0.6b` / `1.8b` / `4b`.

### Оценка

```bash
python pipeline/evaluation/evaluate_unified.py \
    --model-path models/qwen3_8b_sid \
    --data-dir data
```
