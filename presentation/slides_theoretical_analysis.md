# ТЕОРЕТИЧЕСКИЙ АНАЛИЗ — слайды для защиты
## Магистерская диссертация: Адаптация LLM для товарных рекомендаций через Semantic ID

Источниковая база: 31 научная работа (2023–2026), включая 12 работ с промышленными A/B тестами от YouTube, ByteDance, Kuaishou, Meta, Spotify, AliExpress, Tencent.

---

# СЛАЙД «Выводы из научных источников»

## Три ключевых вывода из анализа 31 источника

**Вывод 1. Semantic ID — промышленно валидированная технология**
Подход внедрён 8 компаниями с суммарной аудиторией >3 млрд пользователей:

| Компания       | Система         | Результат A/B теста       |
|---------------|-----------------|--------------------------|
| YouTube       | PLUM [5]        | +4.96% CTR (Shorts)      |
| ByteDance     | TRM [27]        | +0.26% активных дней     |
| Kuaishou      | OneMall [25]    | +14.71% GMV              |
| Kuaishou      | OneRec [2]      | +1.68% Watch Time        |
| Meta          | Ads Ranking [28]| +0.15% online metric     |
| AliExpress    | Hi-Gen [21]     | +1.42% GMV               |

**Вывод 2. Двухстадийное обучение — консенсус всех ведущих работ**
PLUM [5]: без pre-training Recall@10 падает на 18%.
OpenOneRec [29]: без Stage 1 AUC падает на 6%.
Все системы используют: (1) инициализацию эмбеддингов → (2) полную дообучку.

**Вывод 3. Существует фундаментальный компромисс качество/стоимость инференса**
Liu et al. [10]: LLM-as-RS на 20% лучше SID-подхода, но генерирует десятки токенов вместо 3–4.
На масштабе миллиардов запросов SID — единственно реалистичный вариант.
