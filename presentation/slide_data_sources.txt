СЛАЙД. Источники данных для исследования

Amazon Pet Supplies (Amazon Reviews 2023, McAuley Lab, UC San Diego).
Подмножество корпуса Amazon Reviews, содержащее 63 319 уникальных товаров
для животных и 101 926 пользовательских последовательностей покупок.
Для каждого товара доступны текстовые метаданные: название, описание,
характеристики и категория. Данные используются на всех стадиях исследования:
генерация эмбеддингов и семантических идентификаторов, формирование обучающей
выборки в формате диалогов (title-SID, description-SID, sequential, copurchase,
SID-title) и оценка качества рекомендаций по десяти задачам.

Общие данные для совместного обучения (эксперименты H2, H3).
Для исследования влияния состава обучающей выборки на катастрофическое
забывание используются три источника общих данных:
  fineweb-edu (HuggingFace, 2024) -- образовательный веб-текст,
    отобранный по качеству из Common Crawl
  Sky-T1 (NovaSky, 2025) -- reasoning-intensive данные,
    включающие задачи по математике и коду
  Метаданные товаров (title-SID) -- доменные данные,
    сформированные автором на основе каталога Amazon Pet Supplies

Перечисленные источники используются в различных комбинациях и пропорциях
(0%, 25%, 50%, 65% от обучающей выборки) для проверки гипотез H2 и H3.

WikiText-2 (Merity et al., 2016) используется исключительно для оценки
степени катастрофического забывания через измерение perplexity
и не входит в обучающую выборку ни в одном из экспериментов.
